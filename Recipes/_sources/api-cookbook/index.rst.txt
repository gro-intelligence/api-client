#############
API Cookbook
#############

Whether you are just getting started with the Gro API Client or are an expert user, the code examples in this Cookbook should help you get started on your next project.

In this Cookbook, we have assembled recipes of code. These recipes address some of the most asked questions users have when starting with our API Client. Each recipe can be easily copied and edited to address the problem you are working on more closely. 

The recipes have been organized into key themes of the data workflow.
